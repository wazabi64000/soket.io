Interface utilisateur de la chatbox

Structure de la chatbox

Nous allons créer une interface utilisateur moderne et responsive avec React. L’interface comprend :

  • Une zone d’affichage des messages
  • Un champ de saisie et bouton d’envoi
  • Une liste des utilisateurs connectés (plus tard)

Exemple de style CSS

Le style est simple mais animé, avec transitions douces :


#messages {
  max-height: 300px;
  overflow-y: auto;
  border: 1px solid #ccc;
  padding: 10px;
  animation: fadeIn 0.8s ease forwards;
}

#messages div {
  margin-bottom: 8px;
  padding: 6px 10px;
  border-radius: 4px;
  background-color: #f0f0f0;
  transition: background-color 0.3s ease;
}

#messages div:hover {
  background-color: #d0d0d0;
}

input[type="text"] {
  padding: 10px;
  width: 70%;
  border: 1px solid #aaa;
  border-radius: 4px;
  transition: box-shadow 0.3s ease;
}

input[type="text"]:focus {
  box-shadow: 0 0 8px #4a90e2;
  outline: none;
}

button {
  padding: 10px 15px;
  border: none;
  background-color: #4a90e2;
  color: white;
  border-radius: 4px;
  cursor: pointer;
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: #357ABD;
}

@keyframes fadeIn {
  from {opacity: 0;}
  to {opacity: 1;}
}
